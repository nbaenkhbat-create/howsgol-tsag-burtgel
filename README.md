# Цаг бүртгэлийн систем (Booking System)

Хөвсгөл аймагт зориулсан анхны цаг захиалгын систем. Express + JSON хадгалалт + энгийн vanilla frontend (build хийх шаардлагагүй).

## Суулгах & Ажиллуулах

```bash
npm install
npm start
```

Дараа нь: `http://localhost:3000`

> Dev горим (файл өөрчлөгдөхөд автоматаар restart): `npm run dev`

## 3 төрлийн хаяг (URL Architecture)

| Төрөл | Хаяг | Тайлбар |
|------|------|---------|
| **Глобал Админ** | `/admin-secretify` | Шинэ байгууллага бүртгэнэ |
| **Байгууллагын Админ** | `/[нэвтрэх_нэр]` (ж: `/bayr`) | Цаг хуваариа удирдана |
| **Үйлчлүүлэгчийн нүүр** | `/` | Хайлтаар байгууллага олно |
| **Үйлчлүүлэгчийн захиалга** | `/` → хайлт | Нүүр хуудаснаас байгууллага хайж захиална |

## Глобал Админ нэвтрэх (анхны)

Сервер анх ажиллах үед консол дээр хэвлэгдэнэ:

- Нэвтрэх нэр: `superadmin`
- Нууц үг: `admin123`

> Өөрчлөхийн тулд орчны хувьсагч ашиглана: `SUPERADMIN_USER`, `SUPERADMIN_PASS`, мөн `JWT_SECRET`-ийг production дээр заавал солих.

## Ажиллах урсгал

1. `/admin-secretify` → нэвтэрч → байгууллага (Нэр, Компани, Нэвтрэх нэр, Нууц үг, Утас) нэмнэ.
2. Үүсгэсэн нэвтрэх нэр/нууц үгээр `/[нэвтрэх_нэр]` → нэвтэрч, өдөр сонгож цаг хуваариа тохируулна.
   - **Захиалга авахгүй цаг нэмэх** — цайны/амралтын цаг блоклоно (чагтлаад Хадгалах).
   - **Захиалга авахгүй өдрүүд** — бүтэн өдрөөр хаана.
3. Үйлчлүүлэгч `/` дээр байгууллага хайж → илэрц дээр дарж → сул цаг сонгож, нэр+утсаа бичээд захиална.
4. Захиалга байгууллагын админд шууд харагдана.

## Өгөгдөл

Бүх өгөгдөл `data/db.json` файлд хадгалагдана. Устгахын тулд тэр файлыг арилгана.
